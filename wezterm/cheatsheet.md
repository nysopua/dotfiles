# WezTerm チートシート

## LEADER = CTRL + A

## ペイン操作
| 操作             | キー                |
|------------------|---------------------|
| 水平分割         | CMD + D             |
| 垂直分割         | CMD + Shift + D     |
| ペイン移動       | CMD + Alt + 矢印    |
| ペインズーム     | CMD + Z             |
| ペインを閉じる   | CMD + W / LEADER+X  |

## タブ操作
| 操作           | キー              |
|----------------|-------------------|
| 新規タブ       | CMD + T           |
| タブを閉じる   | CMD + Shift + W   |
| タブ選択       | LEADER + 1-9      |

## ワークスペース
| 操作               | キー              |
|--------------------|-------------------|
| 次のワークスペース | CMD + Shift + N   |
| 前のワークスペース | CMD + Shift + P   |

## 表示
| 操作             | キー              |
|------------------|-------------------|
| 透明度トグル     | CMD + Shift + O   |
| ブラートグル     | CMD + Shift + B   |
| フルスクリーン   | LEADER + ;        |

## その他
| 操作           | キー              |
|----------------|-------------------|
| コピー         | CMD + C           |
| ペースト       | CMD + V           |
| 検索           | CMD + F           |
| QuickSelect    | LEADER + Space    |
| リロード       | CMD + Shift + R   |
| チートシート   | CMD + Shift + H   |
